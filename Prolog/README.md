# Como baixar e executar TTT (Prolog)

* Instalar o interpretador SWI-Prolog na sua máquina. Para mais informações de instalação acesse o seguinte link: https://www.swi-prolog.org/
* Clonar este repositório
* Entrar na pasta "prolog"
* Abrir o terminal dentro dessa pasta e executar o interpretador com:

swipl

* Dentro do interpretador, carregar o arquivo principal com:

[main].

* Rodar o jogo utilizando:

main.

⚠️ Aviso: o temporizador pode apresentar falhas em sistemas Linux, mas funciona corretamente em Windows.
